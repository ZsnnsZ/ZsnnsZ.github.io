python数据结构的底层实现

list底层是数组实现，不是链表。
set的底层实现是hash表。
dict的底层实现也是hash表。

在python中要使用链表应该使用标准库collections中的deque，deque是双向链表

判断一个元素是否在某个集合中，list的时间复杂度是O(n),set、dict是O(1)

  • dict有以下几个特点：
    查找和插入的速度极快，不会随着key的增加而变慢；
    需要占用大量的内存，内存浪费多。

  • 而list相反：
    查找和插入的时间随着元素的增加而增加；
    占用空间小，浪费内存很少。

  • set
    是数学意义上的无序和无重复元素集合，交&， 并|

tips

  • 交换两个数字：a, b = b, a
  • 列表推导式：[x for x in range(100) if x%2 == 0]
  • ‘’.join
  • enumerate
  • in/not in
  • dict的get()方法或者setdefault()方法可以传递默认值
  • if is 比 if == 速度快
  • while 1 比 while True 更快
  • 使用**替代pow()，计算更快
